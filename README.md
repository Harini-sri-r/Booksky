# Booksky

Booksky is a beginner-friendly book management app built using HTML, CSS, and JavaScript.

---

## Description
Booksky is a simple book management application that allows users to add, view, and delete books along with the author name and a short summary.  
This project helps beginners understand basic frontend development concepts and JavaScript DOM manipulation.

---

## Features
- Add book title, author, and summary  
- Display added books instantly  
- Delete books easily  
- Simple and user-friendly interface  

---

## Technologies Used
- HTML  
- CSS  
- JavaScript
